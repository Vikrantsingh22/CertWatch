# Project CertWatch — Live Brand Impersonation & Domain Threat Intelligence Platform



## Description

A live system that ingests every SSL certificate issued anywhere on the internet in real time (via Certificate Transparency logs), filters for domains that resemble known brands, enriches each candidate with DNS/WHOIS/hosting/content signals, scores it with an explainable risk engine, clusters related domains into "campaigns," and surfaces all of it on a live operational dashboard — the same shape of system that powers registrar trust & safety teams.
